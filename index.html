HTML Tools

Static HTML and JavaScript tools built with AI.

This repository is a lightweight, static collection of browser-based tools inspired by simonw-tools, but focused on HTML utilities and AI-assisted development.

Each tool lives as a root-level *.html file with an accompanying *.docs.md file. Build scripts generate the homepage and JSON metadata used across the site.
